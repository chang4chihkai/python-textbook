# Python 程式設計教材

**給大一新生的程式設計入門書**

---

## 書籍資訊

- **適用對象**：大學一年級學生 / 程式設計初學者
- **程式語言**：Python 3
- **學習目標**：培養邏輯思維 + 實用程式技能

## 如何使用本書

1. **按順序閱讀**：建議從第 1 章開始依序學習
2. **動手實作**：每章節都有程式碼範例，請務必自己動手輸入執行
3. **完成練習**：每章末的練習題能幫助鞏固所學
4. **專題實作**：完成基礎章節後，可挑戰第 16 章的整合性專案

## 環境建置

請參閱第 1 章了解如何安裝 Python 開發環境。
